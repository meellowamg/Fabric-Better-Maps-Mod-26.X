# Better Maps Mod

A Minecraft 26.1.x Fabric mod that enhances the vanilla map experience with a minimap HUD, improved visuals, and full customization.

Made by Meellowamg

---

## Features

### Minimap HUD
- Pin any held map to your HUD by crouching and right-clicking while holding it
- The minimap stays visible even when you put the map away
- Player marker with real-time rotation and smooth movement
- Arrow icon when on the map, circle icon when off the map
- Switch maps by holding a different map and crouching + right-clicking again

### Visual Enhancements
- Terrain height shading for better elevation visibility
- Water depth shading so you can see how deep the ocean is
- Coastline brightening for clear land/water boundaries

### Configurable via ModMenu
- Map scale (0.5x to 3.0x)
- Map position (top-right, top-left, bottom-right, bottom-left)
- Margin from screen edge
- Border thickness
- Border outer and inner color (hex input with live preview)
- Marker size
- Show/hide player marker
- Show/hide coordinates (X, Y, Z)
- Show/hide current biome name
- Dedicated toggle keybind (set in Controls menu)

---

## How to Use

1. Hold a filled map in your hand
2. Crouch and right-click to pin it to your minimap HUD
3. The minimap appears in the configured corner of your screen
4. Crouch and right-click again (or press your toggle key) to unpin it
5. Open ModMenu → Better Maps Mod → Configure to change settings

---

## Requirements

- Minecraft 26.1.2
- Fabric Loader 0.19.3+
- Fabric API 0.152.1+26.1.2
- ModMenu 18.0.0

---

## Installation

1. Install Fabric Loader for Minecraft 26.1.2
2. Put Fabric API in your mods folder
3. Put ModMenu in your mods folder
4. Put Better Maps Mod in your mods folder
5. Launch Minecraft
